# MillionForge · 财富机会验证操作系统

把长期问题、能力与现实约束转化为可验证的百万美元级商业机会。

## 核心原则

- 不承诺暴富，只筛选和验证候选机会。
- 没有付费访谈、样例数据、第二次会议、订金或试点意向的想法，不算商业证据。
- 优先从已有的一线认知、工程能力和交付经验出发。
- 先出售人工结果，再自动化重复环节，最后才建设平台。
- 敏感数据只保存在浏览器本地，MVP 不包含服务器数据库和第三方分析代码。

## 功能

- 个人问题 DNA 与非对称资产画像
- 内置候选项目与多维机会评分
- 根据真实问题生成 3 条商业化路径
- 谁付费、损失、价格、壁垒、AI 替代风险分析
- 7 天、30 天、90 天、12 个月路线图
- 验证看板、证据记录、淘汰条件
- 本地导入 TXT / MD / JSON / CSV，并更新问题画像
- Markdown / JSON 导出

## 本地运行

```bash
python -m http.server 8080
```

打开 `http://localhost:8080`。

## 部署

- Vercel：直接导入本仓库。
- GitHub Pages：仓库包含 Pages Actions 工作流，推送到 `main` 后自动发布。

## 隐私

所有用户数据使用 `localStorage` 保存在当前浏览器。不要导入案件材料、身份证号、账号密码、单位秘密或其他敏感数据到公开部署。
